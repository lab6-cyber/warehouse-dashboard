# Дашборд анализа складских операций

## Описание проекта
Интерактивный дашборд для анализа ключевых показателей складской деятельности. 
Разработан с использованием библиотеки Dash на языке Python.

## Состав дашборда
Дашборд включает следующие интерактивные элементы:

1. **Линейный график** — отображает динамику доходов, расходов и прибыли во времени
2. **Круговая диаграмма** — показывает структуру расходов по категориям товаров
3. **Гистограмма** — демонстрирует распределение прибыли по операциям
4. **Таблица данных** — содержит детальную информацию по каждой операции
5. **Выпадающий список** — позволяет выбрать период агрегации для линейного графика

## Установка и запуск

1. Клонировать репозиторий:
```
git clone https://github.com/lab6-cyber/warehouse-dashboard.git
```

2. Перейти в директорию проекта:
```
cd warehouse-dashboard
```

3. Установить зависимости:
```
python app.py
```

4. Открыть браузер и перейти по адресу:
```
http://127.0.0.1:8050
```

## Использование

### Загрузка данных
- При запуске дашборд использует встроенный файл с тестовыми данными
- Для загрузки собственных данных нажмите на область "Перетащите файл или выберите файл"
- Формат CSV-файла должен соответствовать структуре: date, product_category, operation_type, quantity, revenue, cost, profit, employee, warehouse_zone

### Выбор периода анализа
- Используйте выпадающий список в панели управления для выбора периода агрегации (день, неделя, месяц, квартал)
- Линейный график автоматически обновится в соответствии с выбранным периодом

### Работа с графиками
- **Линейный график**: отображает динамику revenue (выручка), cost (себестоимость) и profit (прибыль)
- **Круговая диаграмма**: показывает доли категорий товаров в общих расходах
- **Гистограмма**: отображает частоту распределения значений прибыли

### Работа с таблицей
- Таблица показывает первые 20 записей из загруженных данных
- Доступна фильтрация по любому столбцу (введите текст в поле под названием столбца)
- Доступна сортировка по столбцам (нажмите на заголовок столбца)

## Структура CSV-файла
| Поле | Описание |
|------|----------|
| date | Дата операции в формате ГГГГ-ММ-ДД |
| product_category | Категория товара |
| operation_type | Тип операции (Приемка/Отгрузка/Возврат) |
| quantity | Количество единиц товара |
| revenue | Выручка от операции (руб) |
| cost | Себестоимость (руб) |
| profit | Прибыль = revenue - cost (руб) |
| employee | Сотрудник, выполнивший операцию |
| warehouse_zone | Зона склада |
